# PySide2-QtAds
Python/PySide2 bindings to the [Qt Advanced Docking System](https://github.com/githubuser0xFFFF/Qt-Advanced-Docking-System) library.


# Steps to build from source
1. download qt5 and pyside2
2. build qt 5.15.X
3. build pyside2 5.15.X
4. build PySide2-QtAds with `python setup.py bdist_wheel`
